Lista de Tarefas com Recoil

Projeto desenvolvido para a **Aula 20** do curso de React.

## 🚀 Tecnologias utilizadas
- React 18
- Vite
- Recoil
- JavaScript

---

## 📌 Funcionalidades
- Adicionar nova tarefa
- Marcar tarefa como concluída
- Remover tarefa
- Filtrar tarefas:
  - Todas
  - Concluídas
  - Pendentes

---

## 🧠 Conceitos aplicados
- RecoilRoot como provedor global
- Atom para armazenar estado global
- Selector para filtrar tarefas
- Organização por pastas (atoms, selectors, components)
